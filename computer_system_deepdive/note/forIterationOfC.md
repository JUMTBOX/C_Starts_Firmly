## `for...` Iteration of C
- C의 for 반복문 문법은 다음과 같다.
```text
for(<초기화>; <부울 표현식>; <단계>) {
    <본문>
}
```
- for 반복문의 평가 규칙은 다음과 같다.
```text
1. <초기화>를 처음 반복문에 들어갈 때 한 번만 평가한다.
2. <부울 표현식>을 평가한다. 평가한 값이 0(false)이면 for 반복문을 빠져나온다.
   (즉 프로그램이 반복문 본문의 반복을 완료한다.)
3. 반복문 안의 구문인 <본문>을 평가한다.
4. <단계> 표현을 평가한다.
5. 2단계부터 반복한다.    
```
- C는 `<초기화>`와 `<단계>`부분의 구문 목록을 지원하기 때문에 좀 더 복잡한 for문을 작성할 수 있다.
```C
#include <stdio.h>

int main (void) {
    int i, j;
    
    for(i=0, j=0; i < 10; i+=1, j+=10) {
        printf("i+j = %d\n", i+j);
    }
    return 0;
}
```
